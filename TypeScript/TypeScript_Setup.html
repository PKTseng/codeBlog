      # 設定 TypeScript 開發環境

      # 1. 下載跟安裝 node.js

      到 Nodejs (opens new window) 官網下載 Node.js。

      有分 LTS(穩定版) 跟 Current(最新版) 兩個版本，請選擇 LTS 穩定版。

      安裝完後在終端機輸入下面指令

      node -v
      1

      如果安裝好會顯示版本號，如下圖

      # 2. 在本機上使用 npm 全局安裝 typescript

      npm i -g typescript
      1

      不過我是使用 homebrew，所以指令是

      brew install typescript
      1

      安裝完成後查看一下版本號確認是否有安裝

      # 3. 使用 tsc 語法對 ts 檔案進行編譯

      在終端機，路徑必須是當前檔案的資料夾下 對要編譯的檔案輸入指令：

      tsc fileName.ts
      1

      # 4. 執行第一個 TS

      在檔案夾中新增一個 .ts 檔

      然後在終端機輸入 ts class1 他就會開始編譯，如果沒報錯的話代表成功編譯了。

      編譯完成後再同資料夾下會出現另一個 .js 檔。

      這樣就算完成編譯了。
