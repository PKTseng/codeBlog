# TypeScript 型別介紹（１）

# 基本型別

當宣告變數的後要增加一些規範，這些規範是限定變數的值必須是某個型別， 型別分別有： String、Number、boolean

# String

在命名變數的時候要指定 String 型別給變數，如下

let userName: String
userName = 'ken'

// 直接寫法
let userName: string = 'ken'

# 執行 function 時指定型別

  return a + b
}
