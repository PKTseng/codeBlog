# Vue 2 的響應式問題

# $set

在 vue2 當中，當我們使用官方文件中指定的方法 (opens new window)新增或是刪除物件跟陣列內的屬性時，會無法達到響應式的效果。

官方指定的方法有：push、pop、shift、unshift、splice、sort、reverse。

如果不想透過上述的方法也可以使用官方提供的 $set 。

來看範例：

<h1>Vue 2</h1>
<p>名稱：{{ person.name }}</p>
<p>年齡：{{ person.age }}</p>
<p v-if="person.gender">性別：{{ person.gender }}</p>

<button @click="addGender">加入性別</button>

點擊按鈕之前是尚未加入性別屬性的，點擊按鈕後性別屬性已經被加入了，但是畫面並沒有更新。

為了解決這個問題可以使用官方所提供的 set 方法 (opens new window)。

將 addGender function 用 this.$set 改寫

addGender() {
  console.log(this.person.gender);
  // this.person.gender = 'boy';
  this.$set(this.person, 'gender', 'boy');
  console.log(this.person.gender);
},

這時候就可以使用官方文件提供的 delete 方法 (opens new window)。

將 deleteName function 用 this.$delete 改寫

deleteName() {
  console.log(this.person.gender);
  // this.person.gender = 'boy';
  this.$delete(this.person, 'name');
  console.log(this.person.gender);
},

# 操作陣列資料達到響應式

以上都是對物件的資料做操作，再來介紹操作陣列的資料，這就要回到一開始提到的官方文件中指定的方法 (opens new window)。

<h1>Vue 2</h1>
<p v-for="(item, index) in hobby" :key="index">{{ index + 1 }}. {{ item }}</p>

<button @click="updateHobby">更改興趣</button>

資料確實改變了但是畫面上的值卻沒改變，想解決這個問題可以使用剛剛介紹的 $set 方法。

將 updateHobby function 用 this.$set 改寫

updateHobby() {
  console.log(this.hobby[0]);
  // this.hobby[0] = '繪畫';
  this.$set(this.hobby, 0, '繪畫');
  console.log(this.hobby[0]);
},

除了 $set 方法也可以使用官方指定 (opens new window)的那７種可達到響應式了方法，以下用 splice 示範。

updateHobby() {
  console.log(this.hobby[0]);
  // this.hobby[0] = '繪畫';
  // this.$set(this.hobby, 0, '繪畫');
  this.hobby.splice(0, 1, '繪畫');
  console.log(this.hobby[0]);
},
