# Vue 3 生命週期

Vue3 官網生命週期 (opens new window)

Vue3.x 有向下兼容 Vue2.x 語法，所以 Vue2.x 語法可以在 Vue3.x 中運行，但有些語法已經被取代了

取代項目如下：

  1. beforeCreate 跟 created 被 setup 取代。
  2. beforeDestory 跟 destoryed 在 Vue3.x 無法使用。
  3. Vue3.x 的生命週期會比 Vue2 優先執行。
  4. Vue3.x 的生命週期開頭都帶 on，例如 onBeforeMount、onMounted 等等。

# Vue 2 生命週期

在 Vue3 中使用 Vue2 生命週期。

先創造一個子元件（lifeCycle.vue）並在父層 (App.vue) 引用，如下圖。

以下為 lifeCycle.vue 的程式碼

<h2>{{ total }}</h2>
<button @click="total++">+1</button>

import { ref } from 'vue';

export default {
  name: 'lifeCycle',
    setup() {
      
    let total = ref(10);

    return { total };
  },
  
  // Vue2 lifeCycle
  beforeCreate() {
    console.log('生命週期鉤子 ＝> beforeCreate');
  },
  created() {
    console.log('生命週期鉤子 ＝> created');

  beforeMount() {
    console.log('生命週期鉤子 ＝> beforeMount');

  mounted() {
    console.log('生命週期鉤子 ＝> mounted');

  beforeUpdate() {
    console.log('生命週期鉤子 ＝> beforeUpdate');
  },
  updated() {
    console.log('生命週期鉤子 ＝> updated');
  },
  beforeDestory() {
    console.log('生命週期鉤子: beforeDestory');
  },
  destoryed() {
    console.log('生命週期鉤子: destoryed');
  },
};

# Vue 3 生命週期

再看看 Vue 3 ，要引用要先引入以下 API 。

  1. onBeforeMount
  2. onMounted
  3. onBeforeUpdate
  4. onUpdated
  5. onBeforeUnmount
  6. onUnmounted

setup 不需引用，生命週期相當於 Vue 2 的 beforeCreate 跟 created 並優先於 Vue2 先執行，同時在 setup 裡面無法使用 this。

onErrorCaptured : 子/孫代元件的錯誤被捕獲時觸發。

template 跟上面 Vue 2 一樣，只有 script 內容有變動，同時把 Vue2 Vue3 同時做比較，看看兩者的執行速度，需要注意的是 Vue3 執行生命週期是以 callback function 的方式執行，也相當於掛載之前執行。

import { ref, onBeforeMount, onMounted, onBeforeUpdate, onUpdated, onBeforeUnmount, onUnmounted } from 'vue';

export default {
  name: 'lifeCycle',

  setup() {
    let total = ref(10);

    //   // Vue3 lifeCycle

    console.log('Vue 3 生命週期鉤子 ＝> setup');

    console.log('Vue 3 的 this ＝>', this);

    onBeforeMount(() => {
      console.log('Vue 3 生命週期鉤子 ＝> onBeforeMount');
    });

    onMounted(() => {
      console.log('Vue 3 生命週期鉤子 ＝> onMounted');
    });

    onBeforeUpdate(() => {
      console.log('Vue 3 生命週期鉤子 ＝> onBeforeUpdate');
    });

    onUpdated(() => {
      console.log('Vue 3 生命週期鉤子 ＝> onUpdated');
    });

    onBeforeUnmount(() => {
      console.log('Vue 3 生命週期鉤子 ＝> onBeforeUnmount');
    });

    onUnmounted(() => {
      console.log('Vue 3 生命週期鉤子 ＝> onUnmounted');
    });

    return { total };
  },

  // Vue2 lifeCycle
  beforeCreate() {
    console.log('Vue 2 生命週期鉤子 ＝> beforeCreate');
  },
  created() {
    console.log('Vue 2 生命週期鉤子 ＝> created');
  },
  beforeMount() {
    console.log('Vue 2 生命週期鉤子 ＝> beforeMount');
  },
  mounted() {
    console.log('Vue 2 生命週期鉤子 ＝> mounted');
  },
  beforeUpdate() {
    console.log('Vue 2 生命週期鉤子 ＝> beforeUpdate');
  },
  updated() {
    console.log('Vue 2 生命週期鉤子 ＝> updated');
  },
  beforeDestroy() {
    console.log('Vue 2 生命週期鉤子 ＝> beforeDestroy');
  },
  destroyed() {
    console.log('Vue 2 生命週期鉤子 ＝> destroyed');
  },
};

# 卸載
