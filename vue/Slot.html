# Vue 插槽 (Slots)

當父層資料想渲染道子層上時，但有不想用 props 方式傳遞的話這時候就可以使用 slot 的方式，在過往的開發經驗中 slot 的使用情境比較常用在彈窗功能上。

插槽大致上分成三種：

  1. 插槽 ( Slots )
  2. 具名插槽（Named Slot）
  3. 作用域插槽(Scoped Slot)

# 插槽 ( Slots )

以下用最簡單的例子示範，先在父層匯入並引用子層元件。

<!-- 父層 App.vue-->
<div id="app">
  <Child>
    {{ parentData }}
  </Child>
</div>


<!-- 子層 Component -->
<div>
  <h2>我是子層</h2>
  <slot></slot>
</div>

# 具名插槽 (Named Slots)

具名插槽就是 『 有名字的 slot 』，比較常使用在一個組件內有多個 slot 的時候。

當我在子層使用具名 slot 的時候，父層就要用 template 標籤同時給予該標籤 v-slot:name 屬性，讓父層知道該資料是要送給子層的哪一個 slot 標籤。

<!-- 父層 App.vue-->
<div id="app">
  <Child>
    <template v-slot:header>我是父層的 header</template>
    <template v-slot:main>我是父層的 main</template>
  </Child>
</div>


<!-- 子層 Component -->
<div class="child">
  <h2>我是子層</h2>
  <slot name="header">我是子層預設的 header</slot>
  <slot name="main">我是子層預設的 main</slot>
  
  <slot name="footer">我是子層預設的 footer</slot>
</div>
