# Vue 3 Watch 特性

最近在學 Vue3 的時候發現有很多坑，決定在此記錄下來以免日後開發時踩到。

# Vue 2 watch

先來複習一下 vue2 的 watch 屬性，有兩種寫法：

  1. 簡易寫法
  2. 完整寫法

# 簡易寫法

首先寫出一個簡易的模板來觸發 watch 。

<h2>{{ total }}</h2>
<button @click="addTotalNum">+1</button>
1
2

簡易寫法就單純給一個函式就好。

export default {
  name: 'app',
  watch: {
    total(newValue, oldValue) {
      console.log(`newValue=> ${newValue}`, `oldValue=> ${oldValue}`);
    },
  },
  setup() {
    let total = ref(0);
    
    function addTotalNum() {
      total.value++;
    }

    return {
      total,
      addTotalNum,
    };
  },
};

# 完整寫法

再看看看比較完整的寫法。

<h2>{{ total }}</h2>
<button @click="addTotalNum">+1</button>
1
2

完整寫法的話就不是函示而是物件寫法，而物件內又包還三種屬性，分別為 handler function、immediate、deep。

  • handler function：要處理的程序。
  • immediate：當頁面初始化後就會觸發。
  • deep：深度監視並觀察物件內的屬性是否有變動。

# 1. handler function 屬性

export default {
  name: 'app',
  watch: {
   total: {
      handler(newValue, oldValue) {
        console.log(`newValue=> ${newValue}`, `oldValue=> ${oldValue}`);
      },
    },
  },
  setup() {
    let total = ref(0);
    
    function addTotalNum() {
      total.value++;
    }

    return {
      total,
      addTotalNum,
    };
  },
};

# 2. immediate 屬性

有些值在畫面一載入的時候就要開始監聽，這時後就可以使用 immediate 屬性。

  watch: {
   total: {
      handler(newValue, oldValue) {
        console.log(`newValue=> ${newValue}`, `oldValue=> ${oldValue}`);
      },
      immediate: true,

# Vue 3 watch

因為 Vue 3 是 Composition Api ，要使用 watch 屬性的話要記得先引入。

import { watch } from 'vue'
1

Vue 3 的 watch 屬性有 6 個特性（坑）：

  1. 監視 ref 定義的一個響應式資料。
  2. 監視 ref 定義的多個響應式資料。
  3. 監視 reactive 定義的一個響應式資料的全部屬性。
  4. 監視 reactive 定義的一個響應式資料的某個屬性。
  5. 監視 reactive 定義的一個響應式資料的某些屬性。
  6. 特殊情況

# 1. 監視 ref 定義的一個響應式資料。

<h2>{{ total }}</h2>
<button @click="addTotalNum">+1</button>
1
2

watch 屬性傳入的第一個參數是監視的對象，第二個參數是 return 一個 function 概念像是 Vue 2 的 handler 屬性。

import { ref, watch } from 'vue'; // 記得引入 watch 
export default {
  name: 'app',
  setup() {
    let total = ref(0);

    // watch（監視對象，回調函式）
    watch()(total, (newValue, oldValue) => {
      console.log(`newValue=> ${newValue}`, `oldValue=> ${oldValue}`);
    });

    function addTotalNum() {
      total.value++;
    }

    return {
      total,
      addTotalNum,
    };
  },
};

# 2. 監視 ref 定義的多個響應式資料。

概念跟第一點一樣，只是在傳入的參數必須以陣列方式傳入。

<h2>{{ total }}</h2>
<button @click="addTotalNum">+1</button>

<h2>{{ msg }}</h2>
<button @click="changeMsg">更改訊息</button>

import { ref, watch } from 'vue';
export default {
  name: 'app',
  setup() {
    let total = ref(0);
    let msg = ref('原始為更改訊息');

    // watch 第一個參數以陣列方式傳入
    watch([total, msg], (newValue, oldValue) => {
      console.log(newValue, oldValue);
    });

    function addTotalNum() {
      total.value++;
    }

    return {
      total,
      msg,
      addTotalNum,
    };
  },
};

# 4. 監視 reactive 定義的一個響應式資料的某個屬性。

如果監視 reactive 內的一個屬性，除了上面單一值、物件、陣列以外還必須是函式。

 <h2>Age: {{ object.age }}</h2>

# 5. 監視 reactive 定義的一個響應式資料的某些屬性。

如果 reactive 定義的屬性內有多個要 watch 的話，那概念跟之前 ref 多個參數一樣一樣，吧第一個參數用陣列的方式傳入。

<h2>Name: {{ object.name }}</h2>
<button @click="object.name += '～'">加入波浪符號</button>

<h2>Age: {{ object.age }}</h2>
<button @click="object.age++">age + 1</button>

import { reactive, watch } from 'vue';
export default {
  name: 'app',
  setup() {
    let object = reactive({
      name: 'ken',
      age: 18,
    });

    // 用陣列的方式傳入第一個參數
    watch([() => object.name, () => object.age], (newValue, oldValue) => {
      console.log(`newValue=> ${newValue}`, `oldValue=> ${oldValue}`);
    });

    return {
      object,
    };

};

# 6. 特殊情況

這是 Vue 3 watch 屬性比較特殊的情況，如果只想監視物件裡面的深層屬性的話就要加上 deep 屬性。

<h2>salary: {{ object.job.salary.total }} K</h2>
<button @click="object.job.salary.total++">salary +1</button>
1
2
import { reactive, watch } from 'vue';
export default {
  name: 'app',
  setup() {
    let object = reactive({
      name: 'ken',
      age: 18,
      
      // 深層屬性
      job: {
        salary: {
          total: 10,
        },
      },
    });

    watch(
      () => object.job,
      (newValue, oldValue) => {
        console.log(`newValue=> ${newValue}`, `oldValue=> ${oldValue}`);
      }
    );

    return {
      object,
    };
  },
};
